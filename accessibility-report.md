# Accessibility Audit Report

## Issues Found

1. The page only used an h1 heading and lacked a heading hierarchy.
2. Website links were not fully descriptive.
3. Accessibility review was performed to ensure images included alt text.

## Fixes Applied

- Added h2 headings to organize content sections.
- Improved link text to clearly describe destinations.
- Confirmed image includes alt text: "Kenyan Flag".
- Confirmed the HTML document uses lang="en".
- Confirmed form controls in contact.html have associated labels.

## Final Accessibility Status

The page now follows basic accessibility best practices:
- Proper heading structure
- Descriptive links
- Alternative text for images
- Language declaration
- Accessible form labels

- Manual code review
- Chrome DevTools Lighthouse
